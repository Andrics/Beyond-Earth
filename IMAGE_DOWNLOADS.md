## Instructions

1. Visit each URL above (the "Or visit" links are often more reliable)
2. Look for "Download" or "Full Resolution" buttons on the NASA/Hubble pages
3. Right-click on the high-resolution image and "Save Image As..."
4. Save each image to: `frontend/public/images/premium/`
5. Use the exact filename specified (e.g., `pillars-of-creation.jpg`)
6. After downloading all images, the code will automatically reference them locally

## Alternative: NASA Image and Video Library

If direct links don't work, visit:
- **NASA Image and Video Library**: https://images.nasa.gov/

Search for the specific objects (e.g., "Pillars of Creation", "Jupiter Great Red Spot") and download the high-resolution versions.
